Fonctions prédéfinies
Fonctions prédéfinies Description
SELECT CURDATE() Afficher la date courante : pour la date du jour dans une requête
SELECT CURDATE() #0 Affiche la date courante sans les tirets, pratique pour nommer un ficher
SELECT CURTIME() Afficher l'heure
SELECT NOW Date et Heure
SELECT DATE_ADD('2021-12-06', INTERVAL 31 DAY) Affiche XX jour à la date de départ
SELECT DATTE_FORMAT(NOW(), '%d/%m/%Y - %H:%i:%s') autre exemple : SELECT date_rendu, DATE_FORMAT (date_rendu, '%d/%m/%Y') FROM emprunt Change le format de la date du jour au format fr
SELECT CONCAT_WS('-', 'a', 'b', 'c') WS = With separator utile pour concatener un code postal avec une ville
SELECT TRIM('coucou') Supprime les espaces en debut et en fin de chaine
